pub struct BuilderConnectorWidgetsOnly { /* fields omitted */ }

Degraded version of BuilderConnector that can only be used to get widgets.

After the BuilderConnector connects its signals, they cannot be connected again - so the BuilderConnector is consumed. But the widgets are still accessible with this object.
